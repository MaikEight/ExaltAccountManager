2. What Data We Collect & Why

2.1 Legal Basis for Processing

  • Contractual Necessity - For authentication (Auth0 login), processing payments (Stripe), and settings sync.
  • Legitimate Interest - For security measures, fraud prevention, and anonymized analytics.
  • User Consent - For optional settings sync and opting out of analytics.

2.2 Data We Collect

  • Discord ID (used for authentication via Auth0)
  • Auth0 User ID (authentication metadata storage)
  • Stripe Customer ID (for paying users, stored in Auth0 metadata and our backend)
  • Optional user settings (if sync is enabled, stored in Auth0 metadata)
  • Anonymized analytics (usage statistics, unique session counts, non-identifiable hardware information)

2.3 Payment Information

All payment processing is handled securely by Stripe. We do not store credit card information or billing details.

3. Data Retention & Deletion

  • Auth0 & Stripe metadata - Retained as long as the user has an active account.
  • Anonymized analytics - Stored indefinitely unless a user requests deletion.
  • Deleted accounts - Upon request, all personal data is deleted immediately, except for payment records required by tax laws.
  • Error Logs - Stored client-side and deleted by the app after 90 days upon app start.
  • Audit Logs - Logs for game-account data changes (e.g., fetching/storing new data).
  • Debug Logs - Overwritten each app start.

Users can request data deletion via the in-app settings menu or by emailing forget@maik8.de.

4. Where Data is Stored & Transferred

All user-related data is stored within the European Union on NetCup & Hetzner servers (Germany). Stripe & Auth0 may transfer data outside the EU.

If data is transferred outside the EU, we ensure GDPR compliance through:

  • Standard Contractual Clauses (SCCs) approved by the European Commission.
  • Other GDPR-compliant safeguards where applicable.

5. Your GDPR Rights

5.1 Right to Access & Data Export

Users can request a copy of their stored data in JSON format via the in-app settings or email privacy@maik8.de.

5.2 Right to Deletion

Users can request data deletion via the in-app settings or by emailing forget@maik8.de. Deletion is processed immediately (no grace period). Payment records cannot be deleted due to legal obligations.

5.3 Right to Object & Restrict Processing

Users can opt-out of analytics tracking from within the app.

6. Security Measures

  • SSL encryption (HTTPS) for all communications.
  • OAuth2 authentication for secure login.
  • Locally stored sensitive data is encrypted using the Windows Data Protection API.

7. Privacy Policy Updates & Notifications

The Privacy Policy is available via a dedicated website linked from within EAM. If legally required, major updates will be communicated via an in-app popup. Continued use of EAM implies acceptance of the updated policy.
